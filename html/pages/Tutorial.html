Enzyme digestion based 4C-seq.

Example: In a mouse cell line, HindIII enzyme was used for fragmenting crosslinked chromosomes, and a 4C library was constructed for Pou5f1(Oct4) gene upstream enhancer for Illumina sequencing. Users can click and follow the instructions to test enzyme-based 4C-Seq bioinformatic pipeline.



van de Werken, Harmen JG, et al. "Robust 4C-seq data analysis to screen for regulatory DNA interactions." Nature methods 9.10 (2012): 969-972.



A demo FASTQ file with 10 million reads from single-end sequencing can be downloaded here.
Once you complete downloading the test file, please upload the file and input the parameters on W4CSEQ Server Home Page. (See here for details)
After the submitted job is completed, you will see the result page like this.


Gao, Fan, et al. "Comparative analysis of 4C-Seq data generated from enzyme-based and sonication-based methods." BMC genomics 14.1 (2013): 345.
